Key Features

Provide a high-level target or description (e.g., “move to the red cone”), and let the API decide the best movement strategy.

Quickstart

  1. Sign up for an API key in our Pricing section.
  2. Send a POST request to /v1/predict with: 
    
{
  "image": "data:image/jpeg;base64, ...",
  "goal": "Move to the red cone in the center of the frame"
}
  3. Receive a JSON response: 
    
{
  "command": {
    "move_type": "forward",
    "distance_meters": 2.5,
    "turn_angle_degrees": 0
  },
  "confidence_score": 0.93,
  "metadata": {
    "model_version": "v1.0",
    "timestamp": "2025-01-02T12:34:56Z"
  }
}

That’s it! Your robot is now vision-enabled with just a few lines of code.
